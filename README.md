# Fantasy LLM Project

## Purpose

This project is a structured decision engine for NBA player prop betting.

It is not a prediction bot.
It is not a picks generator.

It is a classification system designed to identify when a betting environment is stable enough to support average level production.

## Core Philosophy

Players hit their averages because their role, minutes, and stat pathways are stable.

This project evaluates:
- Role stability
- Minutes protection
- Stat path reliability
- Environmental consistency

The system classifies players into action tiers rather than predicting outcomes.

## Long Term Vision

This engine will eventually integrate with an LLM to:
- Explain classifications
- Summarize injury ripple effects
- Highlight structural risks
- Enforce guardrails

The LLM will assist the model.
It will not override it.

## Current Focus

- NBA player props
- Parlay discipline
- Decision quality tracking
- Structured review process

This repository represents the foundation of a long term, disciplined betting system.
